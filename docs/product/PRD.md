# Ridgemont AI Academy — Product Requirements Document

## Vision
A production-grade educational platform where 80s/90s generational nostalgia meets modern AI education. Students learn AI, coding, media literacy, and systems thinking through memorable themed mentor personas, hands-on labs, and a real-time current-events intelligence layer.

## Product Principles
1. Education is the mission; nostalgia is the delivery mechanism.
2. Every feature must work — no fake dashboards, no placeholder content.
3. AI outputs must cite sources, admit uncertainty, and adapt to learner level.
4. IP safety is non-negotiable — production mode uses inspired archetypes only.
5. The platform must serve students, parents, schools, and adult learners.

## Target Users
| Segment | Need |
|---------|------|
| Teen students (13-18) | Engaging AI/tech education with relatable framing |
| Parents/Guardians | Visibility into child's learning, trust in safety |
| Gen X / Elder Millennials | Modern AI skills via culturally familiar context |
| Schools & Programs | Deployable classroom tool with admin controls |
| Creators & Builders | AI learning + lab platform for building real things |
| Community Centers | Accessible tech education for underserved groups |

## Core Modules
1. **Campus Home** — personalized dashboard, daily briefing, mentor messages
2. **Academy / Courses** — structured tracks, lessons, quizzes, certifications
3. **AI Teacher Personas** — 10 archetype instructors with full agent capabilities
4. **Lab Systems** — 7+ interactive labs (Prompt, Agent, MCP, RAG, Code, Media, Project)
5. **Current Events + Today** — daily AI/tech explainers with era comparisons
6. **Media Generation Layer** — legally-safe era media discovery and analysis
7. **Maker / Builder Track** — create prompts, agents, apps, workflows
8. **Admin / School OS** — user management, content approval, analytics
9. **Parent / Guardian Portal** — progress summaries, safety status
10. **Content Studio** — lesson authoring, quiz generation, persona assignment

## Learning Tracks
1. AI Foundations (12 lessons)
2. Prompting & Communication (8 lessons)
3. Agent Systems (8 lessons)
4. MCP Tools & AI Infrastructure (8 lessons)
5. RAG & Research Systems (8 lessons)
6. Builder Track (6 lessons)
7. Media Literacy & Culture (6 lessons)
8. Business / Creator Track (6 lessons)
9. Digital Assets / Blockchain (optional, 6 lessons)

## User Roles
- Student, Guardian, Instructor, Content Editor, Admin, Super Admin

## Key Features
- **Explain It Through My Era** — modern concepts explained via 80s/90s analogies
- **Generation Theater** — safe media-learning area with educational context cards
- **Daily Homeroom** — personalized morning briefing from assigned mentor
- **Universal Search** — across lessons, projects, media, events, teachers, labs
- **Evaluation System** — rubrics, feedback, retry loops, evidence of completion

## Non-Functional Requirements
- Sub-3s first meaningful paint
- Streaming AI responses
- WCAG 2.1 AA accessibility
- Mobile-responsive
- Multi-tenant ready (consumer + school deployments)
- Billing-ready architecture (subscriptions, licenses, family plans)

## Success Metrics
- Daily active learners
- Lesson completion rate
- Lab engagement depth
- Current-events read rate
- Project submissions
- Parent portal check-ins
- Source citation accuracy
